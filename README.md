# Analise-Logistica-E-commerce-Olist

# 📊 Análise de Dados - E-commerce (Olist)

Este projeto tem como objetivo explorar dados de e-commerce utilizando Python e Power BI, com foco em entender o impacto da logística na receita e na satisfação do cliente.

---

## 🎯 Objetivo

Analisar a relação entre:
- Prazo de entrega
- Receita
- Satisfação do cliente

---

## 🛠️ Ferramentas utilizadas

- Python (tratamento de dados)
- Power BI (visualização e análise)
- Dataset público Olist (Kaggle)

---

## 📈 Principais análises

### 📦 Visão Geral

![Visão Geral](images/visao_geral.png)

- Receita total superior a R$ 13 milhões
- Baixa taxa de cancelamento
- Crescimento consistente ao longo do tempo

---

### 🚚 Logística e Experiência do Cliente

![Logística](images/logistica.png)

- A maior parte dos pedidos é entregue antecipadamente
- Atrasos impactam diretamente a avaliação do cliente
- Entregas com atraso elevado apresentam queda significativa na satisfação

---

## 💡 Insights

- Existe uma relação clara entre atraso e queda na avaliação do cliente
- A operação logística tende a antecipar entregas, indicando possível margem de otimização
- A satisfação do cliente é sensível a desvios no prazo de entrega
